<h1 className="bulma-title">Norwegian Municipalities</h1>

This is a service for finding Norwegian municipalities by their name or coordinates. The data can expressed as
a [Linked Data](https://en.wikipedia.org/wiki/Linked_data) graph, or a simple [JSON](https://www.json.org/json-en.html)
response.

To learn more about the project, read the [documentation](#documentation) or check out
the [code repository](https://github.com/megoth-capgemini/municipalities-coordinates).
